Key features:
  • Dynamic Book Carousel:
    A responsive image slider that allows users to navigate through the Bridgerton book covers using next and previous buttons. The images transition smoothly, creating an engaging user experience.

  • Interactive Descriptions:
    Each book in the carousel is associated with detailed descriptions that update dynamically as the user navigates through the books. This includes the book title, author, rating, and a brief synopsis, providing users with essential information quickly.

  • Secondary Book Section:
    A grid of book thumbnails at the bottom of the page, which also acts as a navigation tool. Clicking on any book in this section highlights it and syncs it with the main carousel, ensuring consistency and ease of use.

  • Responsive Design: The website is fully responsive, ensuring that it looks and functions well on various devices, from desktops to mobile phones.

  • Custom Styling: The website features custom CSS styling, including typography, color schemes, and layout, all designed to evoke the elegant and historical aesthetic of the Bridgerton series.

Bridgerton Bookstore Project

Project Description: The Bridgerton Bookstore Project is a comprehensive UI design and development project that showcases a themed online bookstore centered around the popular Bridgerton series. This project demonstrates my skills in front-end development, including HTML, CSS, and JavaScript. It highlights my ability to create interactive and visually appealing user interfaces, with a strong focus on user experience and responsiveness.
